# chat-server

🔐 Role‑based authentication – One admin account (NiR/7787), unlimited regular users

📢 Public & private chat – /msg <user> <text> for direct messages

🚫 Admin controls – Kick/ban users, enable/disable public chat, send admin broadcasts

🧑‍💻 Unique usernames – Automatic suffix (_1, _2) if a name is already taken

🛡️ IP banning – Banned IPs cannot reconnect

📋 Built‑in help – /help lists all available commands

💻 Console admin panel – Server operator can run the same commands directly from the terminal
